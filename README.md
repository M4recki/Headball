## Usage

Run the main.py file using Python.Player 1, controlled by the user, can move left (A), right (D), and jump (W). To kick the ball, use the SPACE key. Player 2, controlled by the computer, automatically moves towards the ball and kicks it when he is close. The aim of the game is to score as many goals as possible.
